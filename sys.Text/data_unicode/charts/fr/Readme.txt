Le Standard Unicode, version 17.0 - Tableaux des caractères

Le fichier « CodeCharts.pdf » contient l’ensemble des tableaux de caractères
dans leur version définitive pour Unicode 17.0. Avertissement : la taille du
fichier dépasse 100 mégaoctets. Les tableaux de CodeCharts.pdf ne mettent pas
en évidence les nouveaux caractères.

Polices de caractères

L’apparence des glyphes dans les tableaux des caractères Unicode n’est pas
normative. Des variations considérables peuvent exister d’une police à l’autre.
Les polices utilisées dans ces tableaux ont été fournies au Consortium Unicode
par plusieurs concepteurs de polices, lesquels en conservent la propriété. Voir
la liste disponible sous ce lien : https://www.unicode.org/charts/fonts.html
